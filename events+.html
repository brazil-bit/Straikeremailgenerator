How to install in Gmail

1
Click Copy Signature above.
2
Open Gmail → click the gear ⚙ (top right) → See all settings.
3
General tab → scroll to Signature → + Create new → name it Straiker.
4
Click inside the box → press ⌘V (Mac) or Ctrl+V (Windows). Full signature appears with logo.
5
Scroll down → Save Changes. Done.
💡
Event tip: Before a conference, come back here, toggle on Event Mode, pick the event, re-copy, and paste over your existing signature. Takes 30 seconds.
Outlook: File → Options → Mail → Signatures → New → paste with Ctrl+V in the rich-text editor.
